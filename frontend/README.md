# Frontend

React + TypeScript + Vite UI shell for the Connections puzzle generator scaffold.

## Features

- Load a static sample puzzle
- Generate a demo puzzle from the backend
- Reveal the grouped answers
- Inspect score and trace metadata in developer mode

## Local Development

```bash
npm install
npm run dev
```

Set `VITE_API_BASE_URL` if your backend is not running on `http://localhost:8000/api/v1`.
